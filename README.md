# 🎮 Gadget Launch Page — PS5

A simple, static HTML product launch page for the **PlayStation 5 (PS5)**, showcasing its features, technical specifications, worldwide pricing, and launch history.

---

## 📄 About

This project is a single-page HTML website built to simulate a product launch page for Sony's PlayStation 5. It provides a clean, structured overview of the console's highlights, specs, and regional pricing — similar to what you'd see on an official product announcement page.

---

## 🚀 Features Covered

- **Console Features** — Ultra-fast SSD, Ray Tracing, 4K/HDR gaming, DualSense haptic feedback, 3D audio, PS4 backward compatibility, and PlayStation VR support.
- **Technical Specifications** — CPU, GPU, RAM, Storage, Video Output, Audio, and Connectivity details.
- **Launch History** — Global launch (November 2020) and India-specific launch (February 2021).

## ▶️ Getting Started

Since this is a pure HTML project, no installation or setup is needed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/varundev-exe/gadget_launch.git
   ```

2. **Open the page in your browser:**
   ```bash
   cd gadget_launch
   open gadget_launch.html
   ```
   Or simply double-click `gadget_launch.html` in your file explorer.
